# fb_clone_website

This is a simple Facebook clone website that I created while learning Git and GitHub. The main purpose of this project was to practice version control concepts such as initializing a Git repository, creating commits, working with branches, merging changes, and pushing code to GitHub.

The website is built using HTML and CSS and replicates the basic user interface of Facebook, including its layout and design elements. Through this project, I gained hands-on experience in organizing project files, tracking code changes, and managing a development workflow using Git.

Although this is a beginner-level project, it helped me strengthen my understanding of both front-end web development and Git fundamentals. This project is intended for educational purposes and serves as a milestone in my journey of learning web development and version control.

## Technologies Used

- HTML5
- CSS3
- Git
- GitHub

## Features

- Responsive Facebook-inspired homepage
- Clean and simple user interface
- Well-structured HTML and CSS code
- Version-controlled using Git
- Hosted on GitHub for practice and learning
